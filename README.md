Finance data pipeline for automated market data collection from NSE India. Built with Python (Playwright) and pandas for clean EOD price datasets, supporting trading analytics and backtesting workflows.

npm package for fetching and analyzing Discord guild data (roles, channels, members, emojis, permissions). Built with Node.js and discord.js, featuring CLI interface and multiple output formats.
